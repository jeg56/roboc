Jeux labyrinthe 
1- ouvrir un fenetre Dos
2- positionnez vous le répertoire ou vous avez desipper le fichier labyrinthe.zip
3- exécutez le fichier roboc.py (python roboc.py)
4- jouez !!! 	

Présence de 3 fichiers python : 
	- roboc.py permet de gérer l'interface utilisateur, l'initialisation et le pilotage de la classe carte
	- carte.py définition de la class Carte et des méthodes de représentation
	- deplacement.py extension de la class Carte pour facilité la lecture - contient les méthodes liés aux déplacements
